### 👨‍💻 Обо мне
Я PHP-разработчик, который ценит чистый код, системную архитектуру и атмосферные проекты. Сейчас я фокусируюсь на создании надежных backend-решений и автоматизации окружения с помощью Docker.

- 🔭 **Сейчас работаю над:** [Gothic Catalogue](https://github.com/tonysold/gothic_docker) — современной базой знаний по классической RPG на Native PHP.
- ⚡ **Мой подход:** Минимум магии, максимум контроля над архитектурой и производительностью.
- 🛠 **Интересы:** Docker-контейнеризация, оптимизация SQL-запросов и построение кастомных MVC-решений.
